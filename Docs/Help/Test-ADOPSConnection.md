# Test-ADOPSConnection

## SYNOPSIS
Test if an active connection is established to Azure DevOps.

## SYNTAX

```
Test-ADOPSConnection [<CommonParameters>]
```

## DESCRIPTION
Test if an active connection is established to Azure DevOps.

## EXAMPLES

### Example 1
```powershell
PS C:\> Test-ADOPSConnection
```

Test if an active connection is established to Azure DevOps.
Returns $false if not connected.
Returns $true if connected.
