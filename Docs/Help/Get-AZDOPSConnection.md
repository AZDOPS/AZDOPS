# Get-AZDOPSConnection

## SYNOPSIS
Gets all active AZDOPS connections

## SYNTAX

```
Get-AZDOPSConnection
```

## DESCRIPTION
This commands lists all active AZDOPS connecions.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-AZDOPSConnection

----                           -----
MyOrganization                 {Credential, Default}
```
